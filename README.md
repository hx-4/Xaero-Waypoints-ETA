# Xaero Waypoint ETAs
## Estimated Time of Arrival to your destination

This module renders an estimated time below your Xaero Waypoints In-Game View. 

**You must have Xaero's Minimap, Xaero's Worldmap, and XaeroPlus installed**

This module is very customizable. It comes with presets to give you an introduction to the settings, but you can really change it how you like. You can change where the label is, when it appears, how fast you have to go for it to appear, if it shows you waypoint name and distance, etc.

Known issues:
- If you're in the Nether and you've selected Overworld coordinates in the Waypoints menu, it will still show ETA's for where the Nether waypoints would be, not the Overworld waypoints.
